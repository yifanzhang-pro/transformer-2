###  Residual Stream Duality in Modern Transformer Architectures

The residual stream in Transformers can be viewed through a two-axis framework where sequence position and layer depth provide different pathways for information flow, with causal depth-wise residual attention equivalent to sequence-axis short sliding-window attention.
